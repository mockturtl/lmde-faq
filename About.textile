h2. "gollum":https://github.com/github/gollum

Combines a git-aware web server with a text processor to generate HTML from lightweight, legible markup syntax like "markdown":http://github.github.com/github-flavored-markdown/,  "textile":http://redcloth.org/textile, and "MediaWiki":http://www.mediawiki.org/wiki/Help:Formatting.

If you make a series of related edits, use commit messages of the form @fixup! previous message goes here@.  
This lets @rebase --autosquash@ keep the history clean.  It's especially handy to flatten minor changes.

A read-only version, "smeagol":https://github.com/rubyworks/smeagol, removes the edit functionality and is suitable for public hosting.

Language-specific cheat sheets are available in edit mode, via the *?* button.
